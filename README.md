## 💌 마음 우체국
* 연말/연시 안부 인사에 대한 지루함 및 피곤함을 느끼는 경향이 있음
* 간단한 설문 조사 이후 커스터마이징 안부 인사 카드 생성
* naver clova 를 이용하여 간단한 단어들을 이용하여 적절한 인사말 생성
* 카드 이미지 공유를 통한 안부 인사

2. 결과 예시 확인
3. 안부 인사를 보낼 시즌, 카드 디자인, 감성, 연령대, 포함할 단어, 말투 선택
